text

更新日: 2018.09.11

.text(html)

マッチした要素の文字列を取得、または設定します。

// GET
.text();

// SET
.text(text);

パラメータ

GET

GETはパラメータを受け取りません。 最初にマッチした要素から文字列を取得します。


SET

html : string
設定する文字列。 削除する場合は空文字列("")を渡します。

戻り値

GET

string
最初にマッチした要素の文字列を返します。

SET

u
Umbrellaのインスタンスを返します。

例

メインタイトル(h1)の文字列を取得します。

var title = u('h1').text();

メインタイトル(h1)の文字列を設定します。

u('h1').text('Hello world');

デモ

"デモ用文字列"をクリックすると、 "デモ用文字列"→"文字列を変更"に書き換わります。

デモ用文字列
JavaScript
u('#demo').on('click', function() {
  u(this).text('文字列を変更');
});
HTML
<span id="demo">デモ用文字列</span>

関連項目

.html()マッチしたHTML要素を取得、または設定します。
