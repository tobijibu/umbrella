size

.size(filter)

マッチした要素の矩形オブジェクト(ボックス)を取得します。 このオブジェクトには、height、width、top、left、right、bottom が含まれています。


パラメータ

この関数はパラメータを受け取りません。

.size()

戻り値

object
最初にマッチした要素に関する、以下のプロパティを持ったオブジェクトを返します。
  • left
  • right
  • top
  • height
  • bottom
  • width

例

u('body').size();
// {"left":0,"right":400,"top":0,"height":300,"bottom":300,"width":400}
