scroll

更新日: 2018.09.11

.scroll()

マッチした要素まで(サポートされていればスムーズに)スクロールします。

.scroll()

パラメータ

この関数はパラメータを受け取りません。


例

最初のliまでスクロールします。

u('li').scroll();

クリックで.teamクラスを持ったsectionまでスクロールします。

u('a.team').on('click', function(e){
  e.preventDefault();
  u('section.team').scroll();
});

デモ

"PUSH"ボタンを押すと、 ページタイトルのh1までスクロールします。

JavaScript
u('#demo').on('click', function() {
  e.preventDefault();
  u('.main h1').scroll();
});
HTML
<button id="demo" type="button">PUSH</button>
