first

.first()

マッチした最初の要素を取得します。


パラメータ

この関数はパラメータを受け取りません。

.first();

戻り値

node or false
最初の要素か、存在しない場合はfalseを返します。

例

リストの最初の要素を取得します。

var next = u('ul.demo li').first();

デモ

以下のリストをクリックすると、 リストの最初のliを取得し、console.logに出力します。

  • 1-1
  • 1-2

JavaScript

u('#demo').on('click', function() {
  var first = u(this).find('li').first();
  console.log(first);
});

HTML

<div id="demo">
  <ul>
    <li>1-1</li>
    <li>1-2</li>
    <li></li>
  </ul>
</div>

関連項目

.closest() マッチした要素から最も近い親要素を取得します。

.parent() 全ての親要素を取得します。

.children() フィルタを使って直下の子要素を取得します。
