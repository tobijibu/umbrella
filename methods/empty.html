.empty()

マッチした要素の子要素を削除します。

.empty()

パラメータ

この関数はパラメータを受け取りません。


戻り値

u
空のノードを持ったUmbrellaのインスタンスを返します。

例

全ての.containerから子要素を削除します。

u('div.container').empty();

デモ

"PUSH"ボタンを押すと、リストを消去します。

  • 1-1
  • 1-2
JavaScript
u('button').on('click', function(){
  u('#demo').empty();
});
HTML
<div id="demo">
  <ul>
    <li>1-1</li>
    <li>1-2</li>
  </ul>
</div>

関連項目

.remove() マッチした要素を削除します。
